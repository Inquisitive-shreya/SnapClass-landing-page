# SnapClass Landing Page

A modern and responsive landing page designed to showcase the features of **SnapClass**, an AI-powered attendance and classroom management system.

## 🚀 Live Demo

[View Live Project](https://snap-class-landing-page-zeta.vercel.app/)

## 📌 About the Project

SnapClass Landing Page presents the core features of the SnapClass application, including AI-based attendance, face recognition login, smart classroom workflow, and student-teacher interaction. The website is built with a clean UI to give users a quick overview of the platform and its benefits.

## ✨ Features

- Modern responsive landing page
- AI attendance system showcase
- Student and teacher journey sections
- Feature highlights
- Tech stack section
- Clean and user-friendly UI
- Deployed on Vercel

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Vercel
